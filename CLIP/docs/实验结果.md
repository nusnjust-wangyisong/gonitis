## 三、关键发现

### 创新点一（双分支跨注意力融合）的贡献

- ME：Acc +0.040，KL1 F1 +0.078，KL2 F1 +0.048
- AR：Acc +0.044，**KL1 F1 +0.096**（早期骨关节炎检出率显著提升）
- PV：Acc +0.037，全类别 F1 均有提升

### 创新点二（多尺度局部特征提取）的贡献

- ME：Acc 进一步 +0.013，**KL2 F1 +0.058**（轻度 OA 改善最大）
- AR：Acc 进一步 +0.002，**KL1 F1 +0.084**（累计 KL1 提升达 +0.180）
- PV：Acc 进一步 +0.028，**KL4 F1 +0.061**（重度 OA 改善显著）

### MAE 改善（序数误差）

三个数据集的平均绝对误差均持续下降，说明误分类更多发生在相邻等级之间（KL1→KL2），而非跨等级大幅错误（KL0→KL4），与序数分类任务目标一致。
